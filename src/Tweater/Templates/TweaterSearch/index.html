TweaterSearch is a project to re-rank search results in order to achieve a more directed search.

TweaterSearch (name subject to change) will use key phrases, social check-ins, geocodes, and other criteria to determine which tweets are more "interesting" than others.

This project was built by Samuel Jenning using, among other things, Django, the Tweepy API, Python registration, MySQL, and jQuery. Sam doesn't want any criticism, so email praise only to sam jenning at gmail
